# domestic-flights

Search domestic flights within China — all major cities and regional airports covered. Compare prices across all Chinese airlines with real-time availability.

## Overview

Search domestic flights within China — all major cities and regional airports covered. Compare prices across all Chinese airlines with real-time availability. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install domestic-flights
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/domestic-flights
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

Search domestic flights within China — all major cities and regional airports covered. Compare prices across all Chinese airlines with real-time availability.
